# CPP_Boost_and_Guidelines_Support_library

## To install boost library

### Mac OS
`brew install boost`

## To see all information about installed version:
`brew info boost`

## To install Guidelines Support Library GSL:

### Mac OS
`brew install cpp-gsl`

## To see all information about installed version:
`brew info cpp-gsl`

## In the build directory
`cmake .. && cmake --build . -v `
